# Splitwise Clone - Expense Manager

A simple and responsive Splitwise-inspired Expense Manager web application built using HTML, CSS, and JavaScript. This project helps users track shared expenses, split bills among friends or groups, and manage balances easily.

## Features

* Add and manage expenses
* Split bills between multiple users
* Track who owes whom
* Responsive and clean UI
* Real-time expense calculations
* Simple and beginner-friendly project structure

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```bash
├── index.html
├── style.css
└── script.js
```

## How to Run

1. Download or clone the repository
2. Open the project folder in VS Code
3. Make sure all files are in the same folder
4. Open `index.html`
5. Run using Live Server or directly open in browser
